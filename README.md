# py_TelePhoneBook
이번에도 dic, 클래스, 함수, file io등등의 예제를 이용하여 만든 연습용 프로그램이다.
이름하여 'Just_PhoneBook'

**※Python 3.x에서만 작동합니다. Python 2.x에선 작동을 보장하지 않습니다!!**

##기능:
1. 주소록 추가(dic.update, class IMFORMATION)
2. 주소록 수정(dic.update)
3. 주소록 삭제(del dic[])
4. 주소록 검색
5. 등록된 주소록 확인
6. 주소록 TXT파일로 저장(f.write(), f.open())
